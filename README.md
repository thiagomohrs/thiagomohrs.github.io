# Thiago Mohr - Resume Website

Professional resume website for Thiago Mohr da Silveira, Systems Coordinator and Tech Lead.

## Features
- **Modern Dark Theme**: Custom design with neon accents.
- **GSAP Animations**: Fluid scrolling, revealed text, and custom cursor.
- **Multilingual Support**: Available in English, Portuguese, and Spanish.
- **Dynamic Projects**: Highlighted key projects with achievements and research-backed data.
- **Responsive Design**: Optimized for all devices.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (GSAP, ScrollTrigger, ScrollTo)
- Font Awesome
- Google Fonts (Syne, Inter)
